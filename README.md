 🔘 Button

Projeto **Button** desenvolvido com **HTML e CSS**, com o objetivo de criar botões modernos e reutilizáveis para interfaces web. O foco do projeto é trabalhar estilização, efeitos visuais e organização de componentes.

📌 Sobre o Projeto

O **Button** é um conjunto de botões estilizados com diferentes variações de design. O layout foi criado para ser simples, limpo e fácil de reutilizar em outros projetos.


## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3

## 🎯 Funcionalidades

* Botões estilizados
* Efeito hover
* Layout centralizado
* Design moderno
* Variações de botões
* Estrutura reutilizável
